연필갯수

Q: 연필 1다스는 12자루입니다. 학생 1인당 연필을 1자루씩 나누어 준다고 할 때 N명의 학생수를 입력하면 필요한 연필의 다스 수를 계 산하는 프로그램을 작성하시오.

입력 설명
첫번째 줄에 1000이하의 자연수 N을 입력합니다..
출력 설명
첫번째 줄에 필요한 다스 수를 출력한다.
입력예제
178
출력예제
16

    function solution(num){
        let answer = Math.ceil(num/12);
        return answer;
    }
