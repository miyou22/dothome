Q: 서울시는 6월 1일부터 교통 혼잡을 막기위해서 자동차 10부제를 시행한다. 자동차 10부제는 자동차 번호의 일의 자리 숫자와 날짜의 일의 자리 숫자가 일치하면 자동차의 운행을 금지하는 것이다. 예를 들어 자동차 번호의 일의 자리 숫자가 7이면 7일 17일 27일에 운행하지 못한다. 또한 자동차 번호의 일의 자리가 숫자가 0이면 10일, 20일 30일에는 운행하지 못한다. 여러분들이 일일 경찰관이 되어 10부제를 위반하는 자동차의 대수를 세는 봉사활동을 하려 한다. 날짜의 일의 자리 숫자가 주어지고 7대의 자동차 번호의 끝 두자리수가 주어졌을때 위반하는 자동차의 대수를 출력하는 프로그램을 작성합니다

입력 설명
날짜의 일의 자리 숫자가 주어지고, 7대의 자동차 번호의 끝 두자리가 주어진다.
출력 설명
주어진 날짜와 자동차의 일의 자리 숫자를 보고 10부제를 위반하는 차량의 대수를 출력합니다
입력예제
25 23 11 47 53 17 33
출력예제
3

    function solution(day, arr){
        let answer =0;

        for(let x of arr){
            if( x%10===day){
                answer++;
            }
        }
        return answer;
    }
    arr=[25,28,18,47,53,17,33];
    console.log(solution(3,arr))
